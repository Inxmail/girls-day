# Aufgabenstellungen für den Berufsorientierungstag

## Coding Aufgabe - HTML

### Hobby-Seite
* Erstelle eine Webseite in HTML, die ein persönliches Hobby oder Interesse von dir vorstellt.
* Die Hobby-Seite über einen Link, Button oder ein verlinktes Bild von der Startseite aus erreichbar sein.
* Wie du die Seiten gestaltest, bleibt dabei ganz dir überlassen.


## Coding Aufgabe - Java

### Zahlen-Ratespiel
* Erstelle ein Java-Programm, das eine zufällige Zahl generiert.
* Der Benutzer soll versuchen, diese Zahl zu erraten.
* Nach jeder Eingabe informiert das Programm den Benutzer, ob die gesuchte Zahl größer oder kleiner als die eingegebene Zahl ist.
* Sobald die richtige Zahl erraten wurde, gibt das Programm eine Erfolgsmeldung aus.


### Hangman
* Erstelle ein Java-Programm für das Spiel Hangman.
* Gebe ein Wort vor, welches erraten werden soll. Optional kannst du natürlich zufällig ein Wort aus einer Liste auswählen lassen.
* Erstelle ein Array aus Unterstrichen (_) um dem aktuellen Ratestand abzubilden.
* Es sind maximal 6 fehlerhafte Eingaben erlaubt.
* Solange die Anzahl der fehlerhaften Eingaben nicht erreicht wurden oder Buchstaben noch fehlen wird weitergespielt.
* Sobald das Wort erraten wurde, gibt das Programm eine Erfolgsmeldung aus.
